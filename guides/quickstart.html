快速开始 ​

使用当前库前需对 @antv/x6 有一定了解及使用经验。设计器画布、元素节点等详细操作请参考 @antv/x6 官方文档。

环境说明 ​

  1. vue 版本：>3.2.0
  2. @antv/x6 版本：>2.10.1

安装 ​

shell
# NPM
npm i @antv/x6 --save
npm i @easyui/bpmn-builder --save

# PNPM
pnpm add @antv/x6
pnpm add @easyui/bpmn-builder

组件引入 ​

入口文件中完整引入 BpmnBuilder，直接使用内置的默认编辑器

js
import { createApp } from "vue";
import BpmnBuilder from "@easyui/bpmn-builder";
import App from "./App.vue";

const app = createApp(App);

app.use(BpmnBuilder);
app.mount("#app");
